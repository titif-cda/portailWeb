Association Moto Club Millau Passion

Création d'un Site Web

Moto Club Millau Passion est une association qui a pour activité l’organisation d’évènements autours de la moto.
Elle souhaite bénéficier, d’une application lui permettant de gérer ses activités et de promouvoir celles-ci auprès de son public afin de fédérer ses adhérents et d’attirer de nouveaux membres.
Un site web est réalisé, dans un premier temps, pour répondre aux besoins de l’association puis, une application desktop dans un futur proche.
